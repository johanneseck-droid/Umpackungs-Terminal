# Changelog

Alle wichtigen Änderungen an diesem Projekt werden in dieser Datei dokumentiert.

Das Format basiert auf [Keep a Changelog](https://keepachangelog.com/de/1.0.0/),
und dieses Projekt folgt [Semantic Versioning](https://semver.org/lang/de/).

## [1.0.0] - 2026-04-27

### Added
- ✨ Initial Release: Umpackungs-Terminal Webapp
- 🔍 Barcode-Scanning mit Echtzeitsuche
- 📄 PDF-Einbettung aus Google Drive
- 🖼️ Bildanzeige (Produkt- und Behälterbilder)
- ⚠️ Besonderheiten-Kennzeichnung
- 📱 Responsive Design für Mobile & Desktop
- ⚡ Optimierte Datenbanktsuche mit TextFinder
- 🔄 Fehlerbehandlung & Benutzer-Feedback

### Technical Details
- React 18 Frontend über CDN
- Tailwind CSS für Styling
- Google Apps Script Backend
- Google Sheets als Datenbank
- Google Drive für PDF & Bilder (Base64-Encoding)

### Known Issues
- Keine bekannt

## Versionierungsmuster

`MAJOR.MINOR.PATCH` (z.B. 1.0.0)

- **MAJOR**: Große Funktionsänderungen oder Breaking Changes
- **MINOR**: Neue Features, abwärtskompatibel
- **PATCH**: Bugfixes
